## 安装

### npm

```bash | pure
npm install wf-editor --save
```

## 示例

```bash | pure
# 克隆仓库
$ git clone https://github.com/lingjieee/wf-editor.git

# 切换目录
$ cd wf-editor

# 安装依赖
$ yarn install

# 运行示例
$ yarn start
```
