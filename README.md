## Installation

### npm

```bash | pure
npm install wf-editor --save
```

## Examples

```bash | pure
# Clone the repository
$ git clone https://github.com/lingjieee/wf-editor.git

# Change directory
$ cd wf-editor

# Install dependencies
$ yarn install

# Run examples
$ yarn start
```
