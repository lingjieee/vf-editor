## Installation

### npm

```bash | pure
npm install vf-editor --save
```

## Examples

```bash | pure
# Clone the repository
$ git clone https://github.com/lingjieee/vf-editor.git

# Change directory
$ cd vf-editor

# Install dependencies
$ yarn install

# Run examples
$ yarn start
```
